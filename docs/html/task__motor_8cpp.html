This file contains code for a task to control the motors. This will instantiate each of the three motors and send the newest duty cycle from the controller to each of them separately.

void task_motor ( void *  p_params)

Task which initalized the motors and then takes in speed from the controller.

As designed on our circuit board, the pins for the enable, phase, !SLEEP, and !FAULT pins are defined when the attachMotor function of the DRV8256 class. Once the motors are initialized, this task only gets the newest speed from the controller and tells the motor to move by that much.
