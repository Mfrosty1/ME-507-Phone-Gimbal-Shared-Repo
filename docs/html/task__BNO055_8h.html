void task_BNO055 ( void *  p_params)

Task which produces creates pin objects for the IMU and sends necessary data.

Upon start up, the pins for the SDA and SCL are defined using the Wire library. Once the BNO connection has been made successfully, the current Euler angles are retreived and sent to the shares/controller.
